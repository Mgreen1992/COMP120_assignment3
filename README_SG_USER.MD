# User Instructions – How to Use the Task List

This guide explains how a regular user can view the task list for this project.

## 1. Open the Project on GitHub
Open the project’s GitHub page in your browser.

## 2. Go to the Task List
1. At the top of the page, click the **Issues** tab.
2. You will see a list of tasks with short titles.

## 3. View a Task
1. Click any task (issue) to read more details.
2. Inside each task you will see:
   - A description of the task
   - Comments from the project team
   - Whether the task is Open or Closed

## 4. Understand Task Status
- **Open** = Task is still being worked on.  
- **Closed** = Task has been completed.

## 5. What Users Can Do
Users can:
- View tasks
- Read descriptions
- Follow progress

Users **do not create, edit, or close tasks**.

These instructions help users understand how to view the task list and follow the progress of the project.
